range_wave

A modern Flutter application with real-time communication, live location tracking, Stripe payment integration, notifications, and scalable GetX architecture.

✨ Features 🚀 Flutter + GetX architecture 📍 Real-time location services 🔌 Socket.IO live communication 💳 Stripe payment integration 🔔 Push notification support 🌙 Responsive UI with ScreenUtil 🔐 Environment variable support using dotenv ⚡ Optimized app startup & crash protection 📱 Portrait-only mobile experience 🗺️ Direction & map service integration 💾 SharedPreferences local storage 🔄 Dependency injection using GetX

🛠️ Tech Stack Flutter Dart GetX Flutter Stripe Shared Preferences Socket.IO Firebase Notifications Flutter ScreenUtil DotEnv
